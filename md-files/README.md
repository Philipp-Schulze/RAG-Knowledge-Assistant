RAG Knowledge Assistant
Retrieval-Augmented Knowledge Assistant — Fachpraktikum Softwareentwicklung mit Methoden der KI (Modul 63184), FernUniversität in Hagen, SoSe 2026.
Webanwendung, die Dokumente (PDF, DOCX, TXT) aufnimmt und auf Basis der Inhalte Fragen über ein LLM beantwortet — mit Quellenangaben.
